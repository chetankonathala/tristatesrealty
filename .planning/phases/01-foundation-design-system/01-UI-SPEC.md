### shadcn Initialization Notes

Project is greenfield. During implementation, run:

```bash
npx create-next-app@latest tristatesrealty --typescript --tailwind --app
npx shadcn@latest init
```

Configure shadcn with:
- Style: Default
- Base color: Neutral (will be overridden by custom dark tokens below)
- CSS variables: Yes
- Dark mode: class (set as permanent default via `<html class="dark">`)

After init, override CSS variables in `globals.css` with the color tokens declared in this contract.

### Accent Reserved For (explicit list)

The gold accent (#C9A84C) is reserved ONLY for these elements:

1. Primary CTA buttons (background fill)
2. Active nav link underline / indicator
3. "Featured" or "New Listing" badges
4. Price display on listing cards
5. Logo mark / brand icon
6. Hero section CTA button
7. Selected filter pill background
8. Hover underline on text links (not default state)
9. Loading spinner / progress indicators
10. Star ratings

Gold accent must NOT be used for: body text, large background fills, borders on non-interactive elements, icons that are not actionable.

### State Colors

| State | Color | Usage |
|-------|-------|-------|
| Hover (surface) | `--muted` #1A1A1A | Card hover, row hover |
| Focus ring | `--accent` #C9A84C at 50% opacity | Focus-visible outline on all interactive elements |
| Active / Selected | `--accent-muted` | Selected filter, active tab indicator background |
| Disabled | `--muted-foreground` at 40% opacity | Disabled buttons, inactive inputs |
| Error | `--destructive` #DC2626 | Error messages, invalid input borders |
| Success | `--success` #16A34A | Success toasts, completed steps |

## Animation & Transitions (DS-04)

All animations use Framer Motion v11. Performance budget: no animation may cause CLS > 0.1 or INP > 200ms.

| Animation | Trigger | Duration | Easing | Details |
|-----------|---------|----------|--------|---------|
| Page transition | Route change | 300ms | ease-out | Fade in from opacity 0, translate-y 8px to 0. |
| Scroll reveal | Element enters viewport | 500ms | ease-out | Fade + translate-y 24px to 0. Stagger children by 100ms. Trigger at 20% intersection. `once: true` (do not re-animate on scroll up). |
| Card hover | Mouse enter | 200ms | ease-out | translate-y -2px, border-color transition, subtle shadow increase. |
| Nav background | Scroll past 80px | 200ms | ease-in-out | Transparent to `--card` with backdrop-blur-md. |
| Modal enter | Open trigger | 200ms | ease-out | Overlay fade 0 to 1. Modal: opacity 0 scale 0.95 to opacity 1 scale 1. |
| Modal exit | Close trigger | 150ms | ease-in | Reverse of enter. |
| Mobile menu | Toggle | 250ms | ease-out | Slide from right (translate-x 100% to 0). Overlay fade in. |
| Hero content | Page load | 600ms | ease-out | Staggered: heading (0ms delay), subheading (150ms), CTA (300ms). Fade + translate-y 16px. |
| Skeleton shimmer | While loading | 1500ms | linear | Infinite horizontal gradient sweep. |

**Critical constraint:** All scroll-triggered reveals must use `will-change: transform, opacity` and be GPU-composited. No layout-triggering properties (width, height, top, left) may be animated.

## Accessibility Contract

| Requirement | Implementation |
|-------------|---------------|
| Color contrast | All text meets WCAG AA (4.5:1 for body, 3:1 for large text). Gold (#C9A84C) on dark (#0A0A0A) = 7.2:1 ratio. Off-white (#F5F5F0) on dark (#0A0A0A) = 18.1:1 ratio. |
| Focus indicators | 2px outline in `--accent` at 50% opacity, 2px offset. Visible on all interactive elements via `:focus-visible`. |
| Touch targets | Minimum 44x44px for all buttons, links, and interactive elements. |
| Motion | Respect `prefers-reduced-motion`: disable all Framer Motion animations, use instant state changes. |
| Screen readers | All images have alt text. Icon-only buttons have `aria-label`. Modal focus trap active when open. |
| Semantic HTML | Use `<nav>`, `<main>`, `<footer>`, `<section>`, `<article>`, `<h1>`-`<h3>` hierarchy. |
| Skip link | "Skip to main content" link visible on focus, positioned above nav. |

## Performance Budget (DS-05)

| Metric | Target | Strategy |
|--------|--------|----------|
| LCP | < 2.5s | Hero image: use `next/image` with `priority`, serve WebP/AVIF. Preload above-fold fonts. |
| CLS | < 0.1 | Reserve image dimensions via aspect-ratio. Font swap with size-adjust. No layout-shifting animations. |
| INP | < 200ms | No blocking JS in critical path. Use React Server Components for static sections. Defer Mapbox GL load. |
| Total JS bundle (initial) | < 150KB gzipped | Tree-shake Lucide icons. Lazy-load Framer Motion for below-fold sections. Code-split modal and sheet components. |
| Largest image (hero) | < 200KB | Serve hero as WebP at 1920px max-width, quality 80. Use `next/image` blur placeholder. |
| Font files | < 100KB total | Subset Playfair Display (Latin only, weight 700). Subset Montserrat (Latin only, weights 400 + 600). |
| Lighthouse score | > 90 | Measured on deployed Vercel URL, mobile throttled profile. |
